== Description ==

Bizuno Skins, by PhreeSoft, provides several alternate display skins for the Bizuno Application based on the jQuery EasyUI GUI. Includes several dark mode skins.

* Includes all of the freely available themes from the jQuery EasuUI framework. Additional skins and custom skins are available at the developers Website: https://www.jeasyui.com

== Installation ==

Follow the standard installation procedure for all WordPress apps. Once activated the list of available skins will appear in the Bizuno -> Admin -> Profile screen.

= Minimum Requirements =

* Bizuno Accounting plugin
* PHP   version 5.4 or greater (PHP 5.6 or greater is recommended, tested with PHP 7.4)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended, tested with MySQL 5.6 & 5.7)
* WordPress 4.4+

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bizuno-skins` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
